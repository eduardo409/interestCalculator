# Interest Calculator

Given loan principle and APR in decimel value. The application will return daily, monthly and annual expected interest. 

## Example:

loan_principle = `$12,243`

loan_APR = `12.49%`

Make sure to use **python3**

```
python3 cal.py 12243 0.1249
```
